## Setup
You will need a C compiler.  
### Linux:  
```sudo apt-get install gcc make```  
### Windows (Bash):  
```sudo apt-get install gcc make```
### macOS
```xcode-select --install```

## Install
```sudo make install```

## Run
```mte YOUR_FILE```
